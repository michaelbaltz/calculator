# Project: Calculator App

## Overview
A sample calculator application built with Claude code to help demonstrate the power of coding agents and capabilities of Claude.

## Development Phases

### Phase 1: Basic Calculator
Core arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

### Phase 2: Scientific Calculator
Expand with scientific operations:
- Factorials
- Logarithms (natural log, log base 10, arbitrary base)
- Exponentials (e^x, x^y)

### Phase 3: Graphing Calculator
Add calculus support:
- Simple derivatives (symbolic differentiation)
- Simple integrals (symbolic and numerical integration)

## Tech Stack
- Language: Python, React

## Conventions
- Code Style: follow google style guides 
- Testing approach: unit test, ensure 90% coverage of code base
- Branch naming: use features with conventional commits for each commit
